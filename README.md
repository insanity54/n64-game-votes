# insanity54-game-votes

11ty site that generates a styled game votes table with box art from TheGamesDB.

## Setup

```bash
bun install
```

## Build

```bash
bun run build
```

## Dev server

```bash
bun run start
```

## Color palette

- `#01009a` (1,0,154) - Blue
- `#f5b201` (245,178,1) - Yellow/Gold
- `#e10916` (225,9,22) - Red
- `#329900` (50,153,0) - Green
